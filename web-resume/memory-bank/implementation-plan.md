# 项目实施计划

## 概述

本文档为AI生成的个人3D网站提供详细的实施路线图。每个步骤都小而具体，包含明确的验证方法。优先实现基础功能，完整功能后续添加。

---

## 技术栈确认

- **前端框架**: React 18 + Vite
- **3D引擎**: Three.js + @react-three/fiber + @react-three/drei
- **样式**: Tailwind CSS
- **动画**: Framer Motion
- **后处理**: @react-three/postprocessing

## 阶段一：项目基础搭建（第1-3天）

### 步骤1.1：初始化Vite + React项目

**任务描述**：
1. 使用Vite创建React项目（JavaScript，非TypeScript）
2. 安装所有必要的依赖包

**执行命令**：
```bash
npm create vite@latest . -- --template react
npm install three @react-three/fiber @react-three/drei
npm install framer-motion
npm install react-router-dom
npm install -D tailwindcss @tailwindcss/vite
```

**验证方法**：
1. 运行 `npm run dev`，确认开发服务器启动成功
2. 浏览器访问 localhost，确认页面正常显示
3. 运行 `npm run build`，确认构建无错误

## 功能更新：添加3D模型到首页 (2026-03-13) 🔄

### 任务描述
在首页添加真实的3D模型，替换当前的几何体组合头像。可选方案：
- 从 Poly Haven 下载免费模型
- 使用现有的 ModelLoader 组件加载

### 3D模型来源

#### Poly Haven 免费模型
- 网站：https://polyhaven.com/models
- 授权：CC0/商业免费
- 推荐分类：
  - **decorative**: 装饰品（雕像、艺术摆件）
  - **furniture**: 家具（椅子、灯具）
  - **nature**: 自然元素（植物、树木）
  - **electronics**: 电子设备（电脑、显示器）

#### 推荐模型类型
1. **太空人/宇航员** - 科技感，适合程序员简历
2. **机器人** - 赛博朋克风格
3. **电脑/显示器** - 展示技术背景
4. **游戏手柄** - 展示游戏开发方向
5. **植物盆栽** - 增加生活气息

### 执行步骤

#### 步骤1：选择并下载模型
1. 访问 https://polyhaven.com/models
2. 筛选免费模型（Sort by: Downloads）
3. 选择适合简历风格的模型
4. 下载 GLB 格式（或 GLTF + 纹理）

#### 步骤2：模型优化
1. 使用 gltf-pipeline 压缩模型
2. 检查多边形数量（建议 < 50K）
3. 检查纹理尺寸（建议 < 2048px）
4. 转换材质为 meshStandardMaterial

#### 步骤3：集成到项目
1. 将模型文件放入 src/assets/models/
2. 创建或修改 ModelDisplay 组件
3. 添加动画效果（旋转/悬停）
4. 集成到 HomeScene

#### 步骤4：响应式适配
1. 移动端隐藏或简化模型
2. 添加加载状态
3. 性能检测降级

### 验证方法
1. 模型正确加载显示
2. 动画流畅无卡顿
3. 悬停交互正常
4. 移动端正常运行
5. `npm run lint` 无错误
6. `npm run build` 构建成功
